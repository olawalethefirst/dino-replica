# Chrome Dino Replica
This is a nextjs project that replicates the chrome dino game


## Rules
### Nextjs
Only use client components

### Styling 
Use Tailwind css for styling 

### Tests
Write tests for every feature you write 

### CI
Use husky for pre-coimmit hooks

#### Run this after each feature
After each feature, please run this.
```
    bash
    git add .
    .husky/pre-commit
```
